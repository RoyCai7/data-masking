# SUSE Data Masking Service 5 分钟上手

本文档面向最终使用者，帮助你在 5 分钟内完成首次使用。

---

## 1. 打开系统

访问：

- `http://10.146.15.188:8080`

如果页面打不开，先确认服务可访问。

---

## 2. 准备 API Key

如果你已经拿到 API Key：

1. 点击页面右上角 `Key`
2. 将 API Key 粘贴到输入框
3. 点击 `Save`

如果你还没有 API Key：

- 联系管理员为你创建
- 或请管理员在服务器执行：

```bash
cd /opt/data-masking
python3.11 generate_key.py create --name "Your Name"
```

说明：

- 输入框默认显示内容，方便粘贴检查
- 可以点击眼睛按钮切换显示/隐藏

---

## 3. 上传文件

在首页上传区域：

- 拖拽文件到页面
- 或点击选择文件

支持文件类型：

### 文本文件

- `.txt`
- `.log`
- `.csv`
- `.json`
- `.yaml`
- `.yml`
- `.ini`
- `.conf`

### 压缩包 / 归档

- `.tar.gz`
- `.tgz`
- `.tar.bz2`
- `.tbz2`
- `.tar.xz`
- `.txz`
- `.tar`
- `.zip`

限制：

- 单文件最大 500 MB

---

## 4. 设置白名单（可选）

如果某些内容不希望被脱敏，可填入白名单，多个值用逗号分隔。

示例：

```text
localhost,127.0.0.1,my-test-domain
```

---

## 5. 开始脱敏

点击 `Start Masking`。

系统会自动：

1. 上传文件
2. 创建任务
3. 后台脱敏处理
4. 显示处理结果

---

## 6. 查看结果

处理完成后可以看到：

- 风险分数
- 命中的规则数量
- 命中分布图
- 示例替换内容
- 文件行数 / 文件数量
- 处理耗时

---

## 7. 下载结果

结果页可下载：

- `Download Masked File`：下载脱敏后的文件
- `Download Report`：下载 JSON 报告

---

## 8. 查看自己的历史任务

首页下方会显示当前会话的处理历史。

注意：

- 历史任务与浏览器会话相关
- 尽量在同一浏览器中连续操作

---

## 9. 常见问题

## 9.1 看不到结果或提示未认证

处理方法：

1. 点击右上角 `Key`
2. 重新粘贴有效 API Key
3. 点击 `Save`

## 9.2 文件上传失败

检查：

- 文件是否超过 500 MB
- 文件类型是否受支持
- 网络是否正常

## 9.3 找不到之前的任务

原因通常是：

- 浏览器会话变了
- 本地保存的 `X-Session-ID` 变化了

建议：

- 在同一浏览器里完成上传、查看、下载
